Getting Started

Thank You for being a patron of Datalogics
Our AMI is designed to help companies accelerate their workflow.

  1. Launch the product via 1-click.

  2. The Username for the instance is 'centos'; you can access it via terminal 'ssh -i yourpemfile.pem centos@publicdnsnamehere'
    • The pem file is acquired when you create the AMI.
    • The publicdns is the publicdns of your instance assigned by AWS
  3. Our application is located in '/opt/datalogics/Zapier-integration'. 

  4. Modify 'server.js'. 
    • replace https://datalogics.ngrok.io/resource/ in server.js with your own url.
  5. Once the URL has been changed, restart pm2 to apply changes.

    • pm2 reload 0

  6. Once server has been restarted, check PM2's status with command

      7. pm2 status

  7. Logging
    • Logs are located at

      /home/centos/.pm2/logs/PDFCloud-out.log

API Endpoints

Our AMI Offering allows you to easily transform a wide variety of documents into :

  • /bmp - Convert a PDF file to a ZIP of BMP image files
  • /jpg - Convert a PDF file to a ZIP of JPG image files
  • /tif - Convert a PDF file to a ZIP of TIF image files
  • /png - Convert a PDF file to a ZIP of PNG image files
  • /gif - Convert a PDF file to a ZIP of GIF image files
  • /pdf - Converts files to PDF. Accepts word, excel, powerpoint, bmp, jpeg, png, tiff, postscript, html as inputs

Example Commands

As a web API, the application can be called in multiple ways. Below are two examples using Postman and Curl to show how the API can be called.

Curl

curl -X POST "server_url/pdf" -H "accept: application/Json" -H "Content-Type: multipart/form-data" -F "file="@input_file";type=file_type" -F "filename=input_file"

Where server_url is the url of your server and the file options are filled in with your file's name and type

Postman

The API can also be called with Postman using the same arguements..
